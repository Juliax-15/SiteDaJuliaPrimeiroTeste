O que é JavaScript?

JavaScript é a linguagem de programação mais popular do mundo.

JavaScript é a linguagem de programação da web.

JavaScript é fácil de aprender.

Por que estudar JavaScript?

JavaScript é uma das 3 linguagens que todos os desenvolvedores da web devem aprender:

  1. HTML para definir o conteúdo das páginas da web
  2. CSS para especificar o layout das páginas da web
  3. JavaScript para programar o comportamento das páginas da web
Vantagens Desvantagens
É mais rápido que outras linguagens de programação; A detecção de erros no código é mais difícil;
Não consome recursos do servidor; Nem todos os navegadores e dispositivos são compatíveis;
Tem diversas bibliotecas; Possui poucos recursos nativos na linguagem;
Fornece mais interatividade aos sites; Pode se tornar suscetível às falhas de segurança. Devido à execução livre do script, o navegador pode interpretá-lo de maneira distinta, mostrando resultados diferentes do esperado.
